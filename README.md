# Hapi-Socket-io-Heroku

Backend development boilerplate based on hapi-style yeoman generator.

* Framework: HapiJS
* Transport layer: Socket.io
* Hosting: Heroku

## Getting started

### Prerequisites

* Node.js
* NPM
* MongoDB

## Instalation

`npm install`

## Development

`npm run dev`

## Testing

`npm test`

## Deployment

`git push heroku master`
